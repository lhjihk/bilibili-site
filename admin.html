试听台的歌单。type 选 local 要先在「传文件」里上传 mp3 到 audio/，然后 src 填 audio/文件名.mp3（本地歌有真波形）；选 bilibili 填 BV 号；选 163 填网易云歌曲 id。拖动排序暂不支持，删了重加即可调顺序。

首页「选你的武器」三张大卡 + 影像馆子页「三支主打」共用这份数据，改一处两页同步。封面可填外链（B站封面右键复制图片地址），也可点封面下方「⬆ 上传本地图片」直接传本机图（自动压缩上传、自动填好路径）。换视频后封面/毛笔字标题自动跟着变。

影像档案网格。category 可选：original(原创) / cover(翻弹) / gear(器材) / travel(旅行)。封面可填外链，也可点封面下方「⬆ 上传本地图片」传本机图。

手账文章，两种写法任选：① 直接写——点下面「✍️ 写新笔记」当场写正文（自动建好文件，插图也能直接传，最省事）；② 传 Obsidian 笔记——在「传文件」里上传 .md（双链/高亮语法都认）再回这里的记录里把 file 对上。锁定勾上就封存（读者要口令）；bucket 是分桶（旅行/乐记/随笔…随你定，页面自动生成筛选按钮）。改完都要点右上角「保存并发布」。

网站上每一处文字都在这里。标注「HTML」的可以用标签（如 <em>强调</em>、<br>换行、<i>斜体英文</i>），其余填纯文字即可。改完点右上角保存。
