# 编辑台使用说明（admin.html）

> 隐藏入口：`你的网站地址/admin.html`（没有任何页面链接到它，只有你知道）

## 首次使用

1. 打开 `admin.html`
2. 顶部黄色框粘贴 GitHub Token → 点「连接」
   - token 只存在你这台浏览器的 localStorage 里，不会进网站代码
   - 别人打开这个页面没有 token，只能看不能改
3. 看到「✓ GitHub 已连接」就绪

## 九个标签页

| 标签 | 管什么 |
|---|---|
| 🎵 试听曲目 | 首页试听台歌单（B站BV号 / 本地mp3 / 网易云） |
| ⭐ 三首主打 | 首页三张大卡（封面、风味介绍） |
| 🎬 影像档案 | 视频网格（增删、分类、封面） |
| 📓 手账文章 | 文章元数据：标题/摘要/分桶/坐标/**锁定**，每张卡还有「📝 编辑正文」直接改 Markdown 全文 |
| ✏️ 全站文案 | **网站上每一处文字**，53 个输入框按板块分组，带人话说明 |
| 🖼 背景图 | 首页大背景（自动套紫墨滤镜）/ 手账星空底图 / 关于页照片，上传自动压缩+自动填路径 |
| 🧩 板块管理 | 首页六大板块**拖拽排序**（也有↑↓按钮）、显示/隐藏开关 |
| ⚙️ 站点设置 | 主题色（4个色板）/ 彩蛋歌词 / 手账口令与一键封存 / 平台链接 |
| 📦 传文件 | mp3 → audio/，Obsidian 笔记 → articles/，插图自动压缩 → articles/assets/ |

有改动没保存时按钮会变成「保存并发布 ●」、关页面会拦截提醒。改完 → 右上角「**保存并发布**」→ 1 分钟左右全网生效。
没连 token 时可点「下载配置」，手动把 content.json 放回 `data/` 目录。

## 发文章的两条路

**A. Obsidian 直通车（推荐，批量）**
1. 在 Mac 的 Obsidian 里把笔记放进 `90-网站发布区/<桶名>/`（桶文件夹随便建：修行/旅行/生活/乐记/诗集…）
2. 跟 Claude 说一声「同步手账」→ 自动收录（脚本：`python3 tools/sync_journal.py`）
   - 文件夹名 = 分桶；图片自动压缩；双链/高亮语法自动转换
   - 已在编辑台手工改过的摘要/分桶/锁定**不会被覆盖**
3. 想调整摘要/坐标/锁定，再去编辑台「手账文章」里点点改改

**B. 编辑台上传（单篇）**
「传文件」上传 .md → 会问你要不要自动建档 → 去「手账文章」补齐信息 → 保存

## 上锁

- **锁单篇**：「手账文章」勾「锁定」→ 保存（摘要打码、点开要口令）
- **锁整本**：「站点设置」勾「一键封存整本手账」→ 任何人进手账先过封印之门
- **改口令**：「站点设置」输入新口令 → 保存（默认 `2017`）
- 提醒：仓库公开，懂技术的人翻源码能看到原文，绝密内容别上传

## 批量 AI 编辑

直接对 Claude 下令即可，比如：「发布区所有笔记重新分桶」「摘要全部重写」「把含某关键词的笔记全部上锁」——笔记就挂载在这台机器上，批量改完直接上线。

## 彩蛋

任意页面敲 `metal` 五个字母 → 全站失真震动 + 弹一句你的歌词（歌词在「站点设置」编辑）
